Faça uma API usando express, para controle de bloqueio de CPF.

POST /deny-list/cpf
    Bloqueia o CPF - OK
DELETE /deny-list/cpf
    Libera o CPF da lista - OK
GET /deny-list/cpf                  
    Checa se o CPF esta bloqueado - OK
GET /status
    Retorna o status do servidor
        - A quanto tempo o servidor esta rodando; -- OK
        - A quantidade de consultas realizadas desde quando ele esta rodando; -- OK
        - A quantidade de CPF's na deny list; -- OK

OBSERVAÇÔES:

Os retornos desta API precisam ser no formato JSON. - ok 
Deve-se usar MongoDB para o armazenamento da deny list. - ok 
Deve-se validar se o CPF é válido em cada operação que ele for informado (11 digitos e apenas números) - Ok
